Sobre

Kirka é um jogo de voxel FPS programado pelo Pierre-Éli. (Adivinhando do CrazyGames) o jogo está disponível para ser jogado no próprio website de Kirka, mas também é jogável no CrazyGames. Contas são opcionais, mas não consegue entrar ou fazer uma conta pelo "e-mail", tem de usar uma conexão para fazer um novo ou entrar numa conta. Se não decidir para entrar, consegue jogar como um convidado. O jogo é simples, corra ao redor e mate jogadores; embora existam outros modos de jogo que adicionam mais para apenas matar jogadores.
